STEP 6: Field Testing
Differences between Testing and Field Testing:
  • Field testing is done in an uncontrolled environment, either in a direct usage case, or with data that wasn’t part of the original dataset
  • It is used to see how the model would function with it’s expected workload, and see if the amount of data it would receive in X time too much for the model to handle, requiring multiple instances.
  • It helps find any logistical issues or any final issues in the training, and can help determine the best way to gather new data for it to process, so that for instance, using a JPG instead of a PNG does not cause the model to give wrong predictions
