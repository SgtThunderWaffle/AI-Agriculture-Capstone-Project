STEP 1: Finding a problem to use AI on
When determining if a problem is worth using AI on:
  1. Is it a problem largely involving data processing or collection?
  2. Does the problem involve such a large swathe of data that it is unrealistic for a person to be able to process it in a timely manner?
  3. Is it a recurring problem or a one-time issue?
  4. Does the problem have enough underlying data (thousands of datapoints that can be gathered and processed)?
  5. Would the problems nature ever drastically change, leaving current data irrelevant?
  6. Does the problem have clear different categories/ways to differentiate data points?
